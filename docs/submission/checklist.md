# PromptWars Submission Final Checklist

Ensure every item on this list is completed and verified before hitting "Submit" on Devpost/Hack2Skill.

## Core Requirements
- [ ] **Live Frontend URL**: Verify the Firebase Hosting URL is public and loads without errors.
- [ ] **Live Backend URL**: Ensure the Cloud Run URL is active and returns a 200 OK on the health check.
- [ ] **GitHub Repository**: Repo must be Public.
- [ ] **GitHub README**: The master `README.md` must be polished with the project logo and badges.

## Code & Integration Quality
- [ ] **Gemini AI**: Verify Gemini returns valid JSON explanations.
- [ ] **Decision Engine**: Verify the math routes fans *away* from closed gates.
- [ ] **Simulation Module**: Verify the admin panel successfully forces the AI to recalculate the route live.

## Documentation (The "WOW" Factor)
- [ ] **Architecture Diagram**: Attached to submission.
- [ ] **Judges Guide & FAQ**: Easily accessible for the evaluators.
- [ ] **50 Edge Cases Document**: Included to prove enterprise-grade engineering.

## Media Assets
- [ ] **Demo Video**: 5-minute video recorded, edited, and uploaded to YouTube (Unlisted or Public).
- [ ] **PPT Presentation**: 12-slide Canva/PowerPoint exported to PDF and attached.
- [ ] **LinkedIn Post**: Published and URL copied to submission form.

> If all boxes are checked, you are ready to win. Good luck! 🚀
